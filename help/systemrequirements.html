System Requirements
 
Below is outlined the system requirements to run giLE2d.
NOTE: giLE2D was not designed to run on Windows XP and as such, it is not supported
Minimum Recommended
Operating System Windows 7 (32 and 64 bit)
Windows 8		 N/A
CPU Intel Core i3 or equivalent AMD CPU Intel Core i7 Ivybridge or equivalent AMD CPU
RAM 2 gb 4 gb
Graphics Card nVidia GeForce 9800, AMD Radeon 4000 series nVidia GeForce 500 series or higher, AMD Radeon 5000 series or higher
Resolution 1366x768 (lower may frustrate you!) 1920x1080
Disk Drive SATA II SSD
Disk Space 20 mb 1 gb (plus media)
Internet Connection* DSL/Cable 1.5mbps down DLS/Cable 6 mbps down
* Required for registration and updates, help system updates and external links
